# 🛒 Sistema de Gerenciamento de Mercado

Projeto desenvolvido em **Python** utilizando **SQLite**, com foco no gerenciamento de produtos e controle de estoque.

O sistema foi criado para praticar Programação Orientada a Objetos (POO), manipulação de banco de dados e operações CRUD, simulando o funcionamento de um pequeno sistema de mercado.

---

# ✨ Funcionalidades

* 📦 Cadastro de produtos
* 📋 Listagem de produtos
* 🔍 Busca de produtos por ID
* 🔎 Busca de produtos pelo nome
* ✏️ Edição de produtos
* 🗑️ Exclusão de produtos
* ➕ Entrada de estoque
* 🛍️ Venda de produtos
* 📊 Controle automático de estoque
* ✅ Validação de dados informados pelo usuário

---

# 🛠 Tecnologias utilizadas

* Python 3
* SQLite
* Programação Orientada a Objetos (POO)

# ▶ Como executar

1. Clone este repositório.

2. Entre na pasta do projeto.

3. Execute:

```bash
python main.py
```

O banco de dados será criado automaticamente na primeira execução.

# 🎯 Objetivo

Desenvolver uma aplicação em Python capaz de gerenciar produtos de um mercado, permitindo cadastrar, consultar, editar, excluir produtos, controlar o estoque e registrar vendas utilizando um banco de dados SQLite.
